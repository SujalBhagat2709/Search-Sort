# Search Sort

## Overview

Simple implementations of common searching and sorting algorithms.

---

## Features

* Binary Search
* Bubble Sort

---

## Files

* `binary_search.py` → search element in sorted list
* `bubble_sort.py` → sort list using bubble sort

---

## Usage

```bash id="q3fdst"
python binary_search.py
python bubble_sort.py
```
